# Banking-System-in-Java
Project Title:
Simple Banking System in Java

Overview of the Project:
This is a simple console-based banking system application implemented in Java. It allows users to create a bank account, deposit money, withdraw money, and view account details through a command-line interface. The project demonstrates basic object-oriented programming concepts, such as classes, objects, and method calls.

Features:
1. Create a new bank account with a unique account number.

2. Deposit money into the account.

3. Withdraw money from the account with balance check.

4. Display account details including account number, holderâ€™s name, and balance.

5. Basic error handling for invalid transaction amounts.

Technologies/Tools Used:

1. Java (JDK 8 or above)

2. Command-line interface (console)

3. Any Java IDE or text editor (e.g., IntelliJ, Eclipse, VS Code)

4. Git and GitHub for version control and submission

Steps to Install & Run the Project:

1. Installation:

Install Java Development Kit (JDK) from oracle.com.
Optionally, install an IDE like IntelliJ IDEA or Eclipse for easier editing and running.

2. Download or Clone Project:

Clone the repository:

text
git clone <repository-url>
Or download the ZIP and extract.

3. Compile the Program:

Navigate to the project folder in terminal/command prompt.

Compile the Java file:

text
javac BankingSystem.java
Run the Program

4. Run the compiled program:

text
java BankingSystem

Instructions for Testing:

1. When running, enter the account holder name and initial deposit when prompted.

2. Use the menu options to deposit, withdraw, or view account details.

3. Test depositing valid and invalid amounts (negative or zero).

4. Test withdrawing amounts larger than the current balance to confirm that errors are handled.

5. Exit the program via the menu when done.
